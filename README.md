수백 명의 학생들을 **학과(학부), 성비, 신캠 조, 나이** 등을 종합적으로 고려하여 최적의 조 배정을 진행합니다.

<br>  


---

## ✨ 배정 로직

### **🧬 1. 학과(학부)**
도전탐색과정 학생과 반도체공학과 학생들이 있을 것인데, 이 학생들을 조별로 고르게 배정합니다. <br>
(나중에 혹시나 과가 개편되더라도, 프로그램이 알아서 분류하니 걱정 마세요 :D)

<br>

### **🚻 2. 성비**
조마다 성비가 서로 비슷하게 되도록 배정합니다.

<br>

### **🏕️ 3. 신캠 조**
특정 새터 조에 특정 신캠 조 학생들이 쏠리는 것을 방지하여 배정합니다. <br>
단, 아는 사람이 너무 없는 것을 방지하기 위하여 최대한 적당한 인원 (2~3명)씩 묶어서 배정합니다. <br>
(그럼에도 1명씩 배정되는 새내기가 있을 수 있습니다.)

<br>

### **🔢 4. 나이**
특정 조에 나이가 적거나 많은 사람이 쏠리는 것을 방지하여 배정합니다. <br>
또한, 나이가 많은 새내기의 경우, 최소 한 명 이상의 조장이 해당 새내기보다 나이가 많거나 같게끔 배정합니다. <br>
단, 해당 학생이 새터 전체 조장들보다 나이가 많은 경우, 나이가 가장 많은 조장들의 새터 조 중에 한 조에 배정됩니다.

<br>

## 📁 준비 파일
두 개의 파일이 필요합니다.
* 하나는 조장들의 정보가 담긴 `연도ST_leader.xlsx`,
* 다른 하나는 새내기들의 정보가 담긴 `연도ST_freshmen.xlsx`가 필요합니다. <br>
<br>
이때, 파일명을 임의로 바꾸지 않게 유의해주세요. <br>

코드가 파일명 탐색을 기반으로 하기 때문에, <br>
`ST_leader.xlsx` 부분이나 `ST_freshmen.xlsx` 부분의 임의로 건드리게 되면 오류가 발생합니다.
<br> <br>
`연도`에는 해당 새내기배움터 행사 연도를 작성하시면 됩니다. <br>
(예: 2026 새내기배움터 = `2026ST_leader.xlsx`, `2026ST_freshmen.xlsx`)
<br> <br>
각 파일의 구조는 다음 예시와 같이 준비해주시면 됩니다. <br> <br>

#### **`연도ST_leader.xlsx`**
| 조 번호 | 조장1 이름 | 조장1의 생년 | 조장2 이름 | 조장2의 생년 |
| :---: | :---: | :---: | :---: | :---: |
| 1 | 박지니 | 2004 | 최어스 | 2005 |
| 2 | 홍길동 | 2006 | 김철수 | 2007 |

<br>

#### **`연도ST_freshmen.xlsx`**
| 성명 | 출신고교명 | 신캠조 | 학과/학부 | 성별 | 전화번호 | 주민등록번호
| :---: | :---: | :---: | :---: | :---: | :---: | :---: |
| 김영희 | 지스트고 | 9 | 도전탐색과정 | 여자 | 01012345678 | 070102-4000000 |
| 이길동 | 과기원고등학교 | 4 | 반도체공학과 | 남자 | 01000000000 | 061225-3000000 |

## 🛠️ 설치 및 실행 방법



1. **저장소 클론**
   ```bash
   git clone https://github.com/kjw3640/GIST_Saeteo_TeamBuilder.git
   cd GIST_Saeteo_TeamBuilder
   ```

2. 라이브러리 설치
    ```bash
    pip install -r requirements.txt
    ```

3. 실행
    ```bash
    python main.py
    ```
